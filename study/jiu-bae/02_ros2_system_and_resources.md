# 2장. ROS2 기반 로봇 개발에 필요한 정보

## 1. 세계 최대의 로봇 오픈소스 생태계

ROS는 특정 기업이 독점적으로 개발·관리하는 플랫폼이 아니라  
**전 세계 개발자가 함께 만들어 가는 글로벌 오픈소스 커뮤니티**입니다.

ROS는 다음과 같은 목표를 지향합니다.

- **글로벌 로봇 커뮤니티의 허브**
- 로봇 공학을 위한 **공개 소프트웨어·하드웨어 플랫폼**
- 연구용 프로토타입부터 산업용 제품까지 포괄하는 **공통 기반**

### ROS 생태계의 주요 참여 주체

- **학생**
  - ROS 튜토리얼과 예제를 통해 로봇 공학 기초를 학습
- **연구자**
  - SLAM, 경로 계획, 인지 등 최신 알고리즘을 구현·공유
- **로봇 스타트업**
  - 최소 인력으로도 수준 높은 로봇 프로토타입을 빠르게 구현
- **대기업**
  - 자율주행, 물류, 제조 등 산업 현장에 ROS2 기반 솔루션을 도입

### 대표적인 ROS 커뮤니티 채널

- **ROSCon**
  - ROS 개발자 컨퍼런스
  - 최신 ROS 기술 동향, 실제 적용 사례, 튜토리얼 세션 제공
- **ROS Discourse**
  - ROS 개발 토론 포럼
  - 워킹 그룹 소식, 패키지 릴리즈, 설계 논의 등 공유
- **GitHub**
  - ROS·ROS2 관련 소스 코드 저장소
  - 패키지 개발, 이슈 관리, PR 리뷰가 활발하게 이루어지는 공간

## 2. ROS 개발 정보 찾는 방법

ROS 커뮤니티는 **방대한 양의 문서와 예제, Q&A**를 제공합니다.  
개발 중 문제를 해결하거나, 새로운 기능을 학습할 때 아래 자료들을 적극 활용하는 것이 좋습니다.

### 1) ROS Discourse

- `https://discourse.openrobotics.org/`
- ROS 공식 커뮤니티 포럼
- 새로운 패키지 발표, 개발 동향, 워킹 그룹 소식 등 생태계 전반의 흐름을 파악하기 적합

### 2) ROS Documentation

- `https://docs.ros.org/`
- ROS2 공식 문서 사이트
- 설치 가이드, 튜토리얼, API 문서, 아키텍처 설계 문서 등을 포함
- 실습을 진행할 때는 **자신이 사용하는 배포판(Jazzy, Humble 등)을 반드시 선택**하고 따라가는 것이 중요

### 3) ROS2 디자인 문서

- `https://design.ros2.org/`
- ROS2의 설계 철학과 의사결정 과정을 담은 아키텍처 문서 모음
- “왜 이렇게 설계되었는가?” 를 이해하는 데 매우 유용
- 미들웨어 선택, 통신 구조, 실시간성, 보안 등 핵심 주제에 대한 배경 설명 제공

### 4) ROSCon 페이지

- `https://roscon.ros.org/`
- 2012년부터 진행된 ROSCon의 발표 자료와 영상 아카이브 제공
- 실제 산업·연구 현장에서 ROS가 어떻게 활용되는지 **사례 중심으로 학습** 가능

### 5) ROS Answers (로봇 Q&A)

- `https://robotics.stackexchange.com/`
- ROS에 특화된 Q&A를 포함한 로봇 개발 질문 게시판
- 빌드 에러, 런타임 에러, 설정 문제 등 실전에서 자주 마주치는 이슈 해결에 유용
- 질문 전에는 **동일/유사 이슈가 있는지 검색**해보는 습관이 중요

### 6) ROS Wiki

- `http://wiki.ros.org/`
- 주로 ROS1 중심의 위키이지만, 여전히 많은 개념 설명과 패키지 정보가 여기에서 출발
- ROS2 문서가 부족한 부분을 **보완 자료**로 참고하면 좋음

### 7) GitHub (ROS2 소스 코드)

- `https://github.com/ros2`
- ROS2의 실제 소스 코드, 패키지, 예제, 이슈 트래커가 관리되는 **개발의 최전선**
- 문서만으로 이해되지 않을 때는 **직접 코드를 읽어보는 것**이 가장 빠른 이해 방법이 될 때가 많음

### 8) ROS 배포판 리포지터리

- `https://github.com/ros/rosdistro`
- ROS1·ROS2 배포판에 포함되는 패키지 목록과 메타데이터 관리
- 특정 패키지가 어떤 배포판에 포함되어 있는지, 어떤 버전이 릴리즈되었는지 확인 가능

### 9) ROS2 패키지 배포 현황

- `https://repo.ros2.org/status_page/`
- ROS2 패키지 빌드·배포 상태를 한눈에 볼 수 있는 페이지
- 릴리즈 계획, 현재 빌드 상태, 지원 아키텍처 등을 확인하는 데 유용

## 3. 정리 및 활용 팁

- **공식 문서(`docs.ros.org`) + GitHub 소스 코드** 조합이 가장 강력한 학습 루트이다.
- 문제가 생겼을 때는
  1. 콘솔 에러 메시지를 그대로 검색해 보고,
  2. ROS Answers / ROS Discourse를 찾아본 뒤,
  3. 그래도 해결되지 않으면 직접 질문을 올리는 흐름이 효율적이다.
- ROS는 “혼자서 모든 걸 해결하는” 시스템이 아니라,  
  **커뮤니티와 함께 문제를 풀어가는 개발 문화** 위에서 성장한 플랫폼이라는 점을 항상 염두에 두면 좋다.
